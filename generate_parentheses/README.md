# Generate Parentheses

<br>

---

<br>

## Desc

Given n pairs of parentheses, write a function to generate all combinations of well-formed parentheses.

<br>

Example 1:
```
Input: n = 3
Output: ["((()))","(()())","(())()","()(())","()()()"]
```
Example 2:
```
Input: n = 1
Output: ["()"]
```

<br>

Constraints:
```
1 <= n <= 8
```

<br>
<br>

## Topic

* Dynamic Programming
* Backtracking (link to [permutation](../Fbonus/permutation))
