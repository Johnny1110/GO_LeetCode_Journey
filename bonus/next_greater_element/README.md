# Desc 

Given a input int array, return an array that contains the first greater element on the right side for each element in the input array. 

<br>

For example:

input: `[2,1,2,4,3]`

answer: `4,2,4,-1,-1]`

<br>

please using Monotonic Stack/Queue to solve this problem.
